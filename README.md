# 🧪 周期表パズル - 元素を学ぼう！

中学生向けの教育的な周期表パズルアプリです。原子番号1番から20番までの元素（水素からカルシウムまで）をドラッグ&ドロップで正しい位置にはめ込んで、楽しく周期表を学習できます。

## 🎮 プロジェクト概要

このアプリは、中学生が周期表の構造を視覚的に理解し、楽しみながら元素の位置を覚えるための学習ツールです。ゲーム感覚で元素の配置を学べるインタラクティブな教材として設計されています。

## ✨ 主な機能

### 現在実装済みの機能

1. **ドラッグ&ドロップ機能**
   - 元素ブロックをドラッグして周期表の正しい位置に配置
   - タッチデバイス（スマートフォン・タブレット）にも対応

2. **視覚的フィードバック**
   - 正しい位置に配置すると緑色に変化
   - アニメーション効果（拡大縮小）で成功を表示

3. **音声フィードバック**
   - 正解時に心地よい効果音が再生
   - 全問クリア時に特別な完了音を再生

4. **元素情報の表示**
   - 各ブロックに原子番号、元素記号、元素名（日本語）を表示
   - 20種類の元素をカバー（H～Ca）

5. **スコア管理**
   - リアルタイムで正解数を表示（0/20 ～ 20/20）
   - 達成度が一目でわかる

6. **リセット機能**
   - いつでもゲームを最初からやり直せる
   - 元素の配置がランダムにシャッフルされる

7. **完了時の祝福メッセージ**
   - 全ての元素を正しく配置すると祝福画面を表示
   - 「もう一度遊ぶ」ボタンで再挑戦可能

8. **レスポンシブデザイン**
   - PC、タブレット、スマートフォンに対応
   - 画面サイズに応じて最適なレイアウトを提供

## 🚀 使い方

### 基本的な遊び方

1. **アプリを開く**
   - `index.html` をブラウザで開きます

2. **元素を配置する**
   - 画面下部の「元素ブロック」から元素をドラッグ
   - 上部の周期表の正しい位置にドロップ

3. **正解の確認**
   - 正しい位置に配置すると緑色に変わり、音が鳴ります
   - 間違った位置に置くと元に戻ります

4. **クリアを目指す**
   - 全20個の元素を正しく配置すると完了！
   - 祝福メッセージが表示されます

5. **もう一度遊ぶ**
   - 「リセット」ボタンまたは「もう一度遊ぶ」ボタンで再挑戦

## 📞 トラブルシューティング

### 音が鳴らない場合
- ブラウザの音量設定を確認
- 一度画面をクリック/タップしてからプレイ（Web Audio APIの仕様）

### ドラッグできない場合
- 最新のブラウザを使用しているか確認
- モバイルの場合はタッチ操作を試す

### レイアウトが崩れる場合
- ブラウザの拡大率を100%に設定
- ブラウザを最新版にアップデート
